# ✅ 사이트 작동 확인 체크리스트

## 🎯 핵심 확인 사항

### 1. 파일 직접 열기 테스트
- [ ] `html/index.html` 파일을 더블클릭
- [ ] 브라우저에서 정상적으로 열리는지 확인
- [ ] 헤더가 표시되는지 확인
- [ ] 푸터가 표시되는지 확인

### 2. JavaScript 작동 확인
- [ ] 브라우저 개발자 도구(F12) 열기
- [ ] Console 탭에서 오류가 없는지 확인
- [ ] 강의 그리드가 로드되는지 확인
- [ ] 애니메이션이 작동하는지 확인

### 3. 페이지 이동 테스트
- [ ] "강의코스" 메뉴 클릭 → `courses.html` 이동 확인
- [ ] "전문강사 성장 프로그램" 클릭 → `instructor-growth.html` 이동 확인
- [ ] "강의신청하기" 클릭 → `apply.html` 이동 확인
- [ ] "강사커뮤니티" 클릭 → `community.html` 이동 확인

### 4. 기능 테스트
- [ ] 메인 페이지: 강의 그리드 아이템 호버 시 HeaderIndicator 표시
- [ ] 강의코스 페이지: 강의 목록 표시
- [ ] 강의신청 페이지: 3단계 폼 작동
- [ ] 커뮤니티 페이지: 탭 전환 작동
- [ ] 로그인/회원가입 페이지: 폼 유효성 검사 작동

### 5. 이미지 확인
- [ ] 메인 페이지 히어로 이미지 표시 (또는 배경색)
- [ ] 각 페이지 히어로 이미지 표시 (또는 배경색)
- [ ] 이미지가 없어도 레이아웃이 깨지지 않는지 확인

## 🔧 문제 발생 시 확인 사항

### 헤더/푸터가 표시되지 않음
1. `js/components.js` 파일이 존재하는지 확인
2. 브라우저 콘솔에서 오류 확인
3. 파일 경로가 올바른지 확인

### 강의 그리드가 표시되지 않음
1. `js/courses-data.js` 파일이 존재하는지 확인
2. 브라우저 콘솔에서 `COURSES_DATA` 변수 확인
3. `js/courses.js` 파일이 로드되었는지 확인

### 이미지가 표시되지 않음
1. `images/hero/main-hero.jpg` 파일 존재 확인
2. `images/pages/` 폴더의 이미지 파일 확인
3. 이미지 경로가 올바른지 확인 (상대 경로)

### 페이지 이동 시 404 오류
1. 파일명이 정확한지 확인
2. 파일 경로가 올바른지 확인
3. 대소문자 구분 확인 (Windows는 대소문자 무시, Linux/Mac은 구분)

## ✅ 최종 확인

모든 체크리스트를 완료했는지 확인:

- [ ] 파일 직접 열기로 작동
- [ ] JavaScript 오류 없음
- [ ] 모든 페이지 이동 정상
- [ ] 주요 기능 작동
- [ ] 이미지 표시 정상 (또는 플레이스홀더)

**모든 항목이 체크되면 사이트가 정상 작동하는 것입니다!** 🎉
