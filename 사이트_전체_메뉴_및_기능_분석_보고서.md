## 사이트 개요

**사이트명**: Irum Academy (이룸아카데미)  
**목적**: AI 실무 역량 강화 및 취업 전략을 제공하는 온라인 교육 플랫폼  
**주요 타겟**: 
- 직무별 AI 실무 역량을 강화하고자 하는 학습자
- 전문 강사로 성장하고자 하는 강사
- 취업 준비생

### 4. 강의 신청 페이지 (`apply.html`)

**3단계 신청 프로세스**:

#### Step 1: 과정 선택
- 강의 선택 드롭다운
- 희망 일정 선택 (jQuery Datepicker)
- 상담 모드일 경우 시간 선택 추가

#### Step 2: 신청자 정보
- 이름, 이메일, 연락처 (로그인 시 자동 입력)
- 소속 (선택)
- 직무 (필수)

#### Step 3: 확인 및 제출
- 신청 정보 확인
- 개인정보 처리방침 동의
- 제출 버튼

**특수 모드**:
- **강사 모드**: `?type=instructor`
- **상담 모드**: `?type=instructor&action=consult`

**데이터 저장**: localStorage에 신청 내역 저장 (관리자 페이지에서 확인 가능)

## 핵심 기능 분석

### 1. 인증 시스템

#### 일반 로그인/회원가입
- **프론트엔드**: jQuery Validation으로 폼 검증
- **백엔드 연동 준비**: API 엔드포인트 정의됨
- **Fallback 모드**: 백엔드 없이도 localStorage로 동작
- **데이터 저장**: 
  - `isLoggedIn`: 로그인 상태
  - `userEmail`: 사용자 이메일
  - `userName`: 사용자 이름
  - `authToken`: 인증 토큰 (백엔드 연동 시)
  - `users`: 회원 목록 (배열)

#### 소셜 로그인
- **지원 플랫폼**: Google, Naver, Kakao, Apple
- **개발 모드**: `OAUTH_DEV_MODE = true`로 시뮬레이션 가능
- **설정 파일**: `js/oauth-config.js`
- **데이터 저장**: 일반 회원가입과 동일한 구조

### 3. 신청 시스템

```javascript

  id: "timestamp",
  courseSlug: "marketing-ai",
  courseTitle: "AI 마케팅 실무",
  preferredSchedule: "2024년 3월 15일",
  preferredTime: "14:00", // 상담 모드일 경우
  name: "홍길동",
  email: "user@example.com",
  phone: "010-1234-5678",
  organization: "회사명",
  position: "마케터",
  applicationType: "student" | "instructor" | "consult",
  submittedAt: "ISO date string",
  status: "pending" | "approved" | "rejected"

```

1. 과정 선택 → 2. 신청자 정보 입력 → 3. 확인 및 제출

- localStorage에 저장 후 관리자 페이지에서 확인 가능

  - `isLoggedIn`: 로그인 상태
